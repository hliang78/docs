# OneOPS RCA 真实验收记录模板

本文档只用于未来真实数据验收时直接填报。

当前结论先固定为：

- 代码侧基线夹具已通过
- 真实数据执行当前延期
- 在真实数据窗口开启前，不扩展新的 RCA 能力面

使用要求只有四条：

1. 一次记录只对应一个真实租户
2. 一次记录只对应一组同租户告警
3. 一次记录只对应一个观测时间
4. 若拓扑输入不满足最小契约，当次记录直接结束
