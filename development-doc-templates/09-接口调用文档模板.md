## 4. 状态码说明

| 状态码 | 说明 | 常见原因 |
|--------|------|----------|
| 200 | 成功 | 请求处理成功 |
| 400 | 参数错误 | 必填参数缺失、格式错误、长度超限 |
| 401 | 未授权/未登录 | Token 缺失或无效 |
| 403 | 无权限 | 当前用户无操作权限 |
| 404 | 资源不存在 | ID 不存在或资源已删除 |
| 500 | 服务器内部错误 | 服务异常 |
| 601 | 权限不足 | 业务权限不足 |
